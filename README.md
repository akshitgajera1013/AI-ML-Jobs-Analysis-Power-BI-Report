# AI-ML-Jobs-Analysis-Power-BI-Report

This project presents a Power BI Report that analyzes trends in the Artificial Intelligence and Machine Learning job market. The report provides insights into job demand, required skills, experience levels, salary distribution, and job locations using interactive Power BI visuals.

🛠 Tools & Technologies
  Power BI Desktop
  Power Query
  DAX
  Data Cleaning & Data Modeling
