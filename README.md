AI & ML Jobs Market Analysis Dashboard

Project Overview

This project analyzes the Artificial Intelligence and Machine Learning
job market using a structured dataset containing information about job
roles, salaries, required skills, companies, and locations.

The goal of this analysis is to uncover insights about salary trends,
skill demand, job distribution, remote work patterns, and hiring trends
in the AI/ML industry.

Using Power BI, an interactive dashboard was developed to visualize
relationships between job roles, required skills, experience levels,
salary ranges, and company hiring behavior.

The dashboard helps answer key questions such as: - Which AI/ML roles
offer the highest salaries? - What skills are most demanded in the job
market? - How does experience level impact salary? - Which companies are
actively hiring AI/ML professionals? - What is the trend of remote jobs
in AI/ML?

This project demonstrates data cleaning, relational modeling, and
interactive data visualization skills using Power BI.

------------------------------------------------------------------------

Dataset Overview

The dataset consists of multiple related tables representing different
aspects of the AI/ML job ecosystem.

Table Description Jobs - Contains job title, salary, experience level,
remote ratio, and job posting details Companies - Information about
companies posting AI/ML jobs Locations - Geographic details such as city
and country Skills - List of technical skills required for AI/ML roles
Job_Skills - Mapping table connecting jobs with required skills

These tables are connected through relational keys to create a data
model for analysis in Power BI.

Key Dataset Features

The dataset includes important attributes such as:

-   Job Title
-   Salary (USD)

-   Remote Ratio
-   Company Name

These features allow analysis of:

-   Salary distribution
-   Skill demand trends
-   Experience level impact

-   Company hiring behavior

Dashboard Overview

The Power BI dashboard is divided into multiple analytical sections to
explore the AI/ML job market from different perspectives.

1. Overview Dashboard

The Overview section provides a high-level summary of the dataset
including:

-   Total number of jobs
-   Average salary
-   Median salary
-   Remote job percentage
-   Average salary by experience level

Additional visualizations show: - Job postings by month - Experience
level distribution - Country and city-based job counts

------------------------------------------------------------------------

2. Location Insights

This section analyzes how job opportunities and salaries vary across
different locations.

Key insights include: - Job distribution across cities - Country-wise
job availability - Salary variation by location

------------------------------------------------------------------------

3. Company & Role Insights

This section focuses on company hiring trends and job roles.

Visualizations include: - Jobs per company - Job titles by experience
level - Company ratings - Salary distribution across job roles - Remote
ratio vs salary

------------------------------------------------------------------------

4. Skill & Hiring Insights

This section focuses on the demand for technical skills.

Visualizations include: - Skill usage across experience levels - Skill
demand distribution - Skills associated with higher salaries -
Experience level vs skill counts

------------------------------------------------------------------------

Key Insights from the Analysis

-   Machine Learning Engineers and Deep Learning Specialists tend to
    have higher salaries.
-   Skills like Python, Deep Learning, NLP, and Cloud Computing are
    highly demanded.
-   Many AI/ML jobs offer remote or hybrid work options.
-   Experienced professionals receive higher salaries than entry-level
    roles.
-   Major companies such as OpenAI, NVIDIA, and Google DeepMind are
    active in AI hiring.
